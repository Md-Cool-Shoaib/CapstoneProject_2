# BankAccount Java Project

This project is part of my **Week 2 of learning Java**. It demonstrates basic object-oriented programming concepts by simulating a simple bank account system.

## Features

* Create account with minimum balance validation
* Deposit and withdraw money
* View account details
* Track total bank money using a static variable

## Concepts Used

* Classes & Objects
* Constructors
* Encapsulation
* Static variables
* Basic validation logic

Built to strengthen my Java fundamentals while progressing toward becoming a **Java Developer**.
